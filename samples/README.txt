================================================================================
  Object Detection Dataset - Sample Images
================================================================================

This folder contains sample images used for the dataset preview page.

Required Sample Images:
- sample1.jpg (Main demo image with multiple objects)
- person_sample.jpg (Person detection sample)
- car_sample.jpg (Car detection sample)
- dog_sample.jpg (Dog detection sample)
- cat_sample.jpg (Cat detection sample)
- bicycle_sample.jpg (Bicycle detection sample)
- motorcycle_sample.jpg (Motorcycle detection sample)

Image Specifications:
- Recommended resolution: 640x480 or higher
- Formats: JPG, PNG
- Should contain clear examples of the object classes
